INTERVIEW DEMO SCRIPT
================================================================================

1. Open http://localhost:8001

2. Say: "Let me show you how LangChain processes natural language"

3. Click: "Show me all sensors"

4. Point out each step as it appears:
   - Step 1: Tokenization - Breaking text into words
   - Step 2: Intent Recognition - Understanding what user wants (95% confidence!)
   - Step 3: Entity Extraction - Finding specific items (sensor IDs, etc.)
   - Step 4: API Mapping - Automatically choosing the right endpoint
   - Step 5: API Execution - Calling the backend
   - Step 6: Response Formatting - Making it user-friendly

5. Say: "Notice the total time - 225ms. This is why it feels instant to users"

6. Click another query: "Get sensor-1 details"

7. Say: "See how it extracted 'sensor-1' automatically? That's entity extraction"

8. Explain the benefit:
   "This reduced our development time by 20-30% because developers don't 
    need to write routing logic. LangChain handles it automatically."
